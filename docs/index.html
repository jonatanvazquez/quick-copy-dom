Quick Copy DOM

Select any element on a webpage and instantly copy its HTML to clipboard. Like DevTools inspector, but faster.

Add to Chrome View on GitHub

Features

Visual Highlighting

Elements highlight as you hover, just like Chrome DevTools inspector. See exactly what you're selecting in real-time.

One-Click Copy

Click any element to instantly copy its complete outerHTML to your clipboard. No menus, no extra steps.

Keyboard Shortcut

Quick access with Ctrl+Shift+E (Cmd+Shift+E on Mac). No need to click the extension icon every time.

Element Info

Tooltip shows tag name, CSS classes, and dimensions while hovering. Know exactly what you're copying.

Lightweight

Minimal footprint, no bloat. Only activates when you need it, doesn't slow down your browsing.

Privacy First

No data collection, no tracking, no analytics. Everything happens locally in your browser.

How It Works

1

Activate

Click the extension icon or press Ctrl+Shift+E to start selecting

2

Hover

Move your mouse over the page. Elements highlight as you hover.

3

Click

Click on any element to copy its HTML to your clipboard instantly.

Keyboard shortcut: Ctrl + Shift + E (or Cmd on Mac)
